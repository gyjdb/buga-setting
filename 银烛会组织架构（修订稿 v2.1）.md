## 二、内设部局〔一厅三局两署一处〕

> **〔命名原则〕** 银烛会之内设机构依职能性质采用不同名称：承担核心业务且编制较大者称"总局"或"局"；承担专项审查与评估者称"署"；承担内部管理者称"处"；综合枢纽称"办公厅"。各机构之下设单位依层级称"司""处""科"或以职能直接命名。

### 4. 外域情报局

银烛会之对外情报中枢。涵盖混沌力量、凡世诸国、黑暗势力与元素疆界之全方位情报收集与分析。

| 下设单位 | 职能 |
|---------|------|

| 黑暗边境监视司 | 玛达拉等黑暗势力之专项监视——军力变动、黑暗巫师活动、边境态势 |
| 凡世态势监视司 | 凡世诸国军事与政治动向——战争、圣战、帝国更迭等可能波及联盟之事态。严格遵守最低接触原则 |

| 情报综合研判室 | 各来源情报汇总、交叉验证与综合态势评估报告编写 |

### 5. 防务监察局

银烛会之独立监察机关。对舰队、城邦卫戍军及一切涉及联盟防务之机关与人员进行合规监察。监察报告直通议会，不经被监察对象审批。

| 下设单位 | 职能 |
|---------|------|
| 舰队监察司 | 对银色舰队之合规监察——军令合法性、军纪、军官任免、军事预算执行 |
| 城邦卫戍军监察司 | 对各城邦卫戍军之合规监察——最低标准达标、编制是否超标、征召配合 |
| 船坞与军工监察司 | 军用船坞产能基线、委托运营契约执行、军工质量 |
| 禁术使用监察科 | 军事领域之禁术使用专项监察——禁咒授权执行是否越界、军事研究违禁实验 |
| 监察通报与报告司 | 监察报告之起草、审核与发布——得直接呈送议会 |
| 举报受理与保护科 | 联盟公民与军事人员对防务违规之举报受理、线索分发、举报人保护 |

### 6. 联盟战备审查署

独立于防务监察局之常设评估机关。监察局管"查违规"（有没有人犯错），审查署管"查战力"（能不能打赢）——前者是纪检，后者是体检。

| 下设单位 | 职能 |
|---------|------|
| 舰队战备评估司 | 定期评估各舰队群之战备等级、人员满编率、装备完好率、训练达标率 |
| 城邦卫戍军评估司 | 评估各城邦卫戍军是否符合最低标准与上限标准 |
| 船坞产能核验科 | 军用船坞产能基线之年度核验——与舰务行政部联合执行 |
| 联合演习效能评估科 | 联盟级联合演习之实效独立评估 |
| 战备总报告室 | 各项评估汇总，编写年度战备审查报告呈送议会 |

### 7. 内部合规处

银烛会之自我监督机关。——"谁来监督监督者"的制度回答。编制精小，但权限锋利。

| 下设单位 | 职能 |
|---------|------|
| 内部纪律科 | 银烛会人员之纪律审查与违规调查 |
| 信息安全与反渗透科 | 内部情报泄露防范、人员背景复查、安全审查 |
| 外部移送科 | 严重违规案件之调查整理与移送裁断体系或盟约监察院 |

> 内部合规处由副总监（监察与审查）直管。涉及副总监本人之案件由总监亲自处理；涉及总监之案件直接移送盟约监察院。

## 六、职级体系

### 6.1 银烛会职级总表

银烛会职级分为**四等十二级**，与联盟公务体系对等但独立编号，以维护银烛会之独立身份。

|----|------|------|------------|------|
| **特等** | 特级 | **银烛特监** | 总监 | 仅一人 |
| **首等** | 正一级 | **银烛大监** | 副总监、总秘书长 | |
|  | 副一级 | **银烛上监** | 总局局长、局长、署长 | |

|  | 副二级 | **银烛副监** | 副司长、中型分区主管 | |

### 6.2 职级说明

1. **特等·银烛特监**——仅总监一人享有此级，对应联盟最高级公务员。
2. **首等**——银烛会之高级领导层。正一级对应副总监与总秘书长，副一级对应各总局局长、局长与署长。
3. **中等**——银烛会之中坚力量。从副局长到各类科长、堡垒指挥官、高级专业人员。此等人员为银烛会日常运转之骨干。
4. **基等**——银烛会之基层人员。从主任科员到学员。基等人员数量最多，分布于台网分区、边境堡垒与各内设单位。

### 6.3 晋升规则

1. 六级（学员）至副四级之晋升以培训合格与年限为主。
2. 正四级以上之晋升须经银烛会人事考核，以专业能力、工作实绩与纪律表现为综合评定。
3. 副一级以上之晋升须经总监提名、议会确认。
4. 跨等晋升（基等→中等、中等→首等）须有特殊贡献或紧缺岗位之需要，经总监办公会合议。
5. 银烛会之人事考核由总监办公厅秘书司统筹组织，内部合规处享有监督权。

### 6.4 特殊身份

除职级体系外，银烛会另有下列特殊身份：

| 身份 | 说明 |
|------|------|
| **银烛密使** | 非常设职级，由总监临时任命。密使持密使令执行专项任务，任务期间享有跨级权限。任务结束后恢复原职级 |
| **银烛密探** | 驻外暗哨之情报人员。持有正式职级但身份不公开，对外以凡世身份活动 |

| 等级 | 色标 | 名称 | 触发条件 | 发布范围 | 后续程序 |
|------|------|------|---------|---------|---------|
| **五级** | 白 | **银烛白烟** | 节点波动在历史区间内但出现非常规模式；边境出现异常但性质不明 | 银烛会内部（相关分区与总局） | 加强监测，指定专人跟踪，编写内部跟踪简报 |
| **四级** | 蓝 | **银烛蓝焰** | 节点异常趋势持续且呈恶化迹象；边境确认出现值得关注之动向；凡世事态可能影响联盟 | 银烛会全会 + 相关对口机关（节点与结界部、最低接触外务部等） | 成立专项跟踪组，向对口机关发送通报函 |
| **三级** | 黄 | **银烛黄光** | 节点异常超出历史区间但尚未危及结构稳定；边境发生可确认之小规模混沌渗透或敌对试探；魔潮早期预兆出现 | 银烛会 + 议会元老会 + 舰队统帅部 | 议会元老会评估是否通报两院；统帅部提升相关方向战备等级 |
| **二级** | 橙 | **银烛橙炬** | 节点出现结构性异常且有崩坏趋势；边境发生较大规模之混沌突破或敌对行动；魔潮预兆明确 | 议会两院 + 统帅部 + 执政院 + 相关城邦 | 议会评估是否宣布紧急状态或启动戒备动员 |
| **一级** | 赤 | **银烛赤焰** | 节点即将崩坏或已崩坏；混沌大规模突破已发生或迫在眉睫；联盟空域遭受直接攻击；Tiamat法则发生重大变轨 | **同时通报议会、统帅部与银色学派——不受任何常规程序限制** | 议会紧急会议；可能触发红线事件流程；银色学派评估是否发布接管令 |

### 预警升降级规则

1. 预警等级之升级由银烛总值班室依实时数据初步判断，经当值高级官员（正二级以上）确认后即时发布。五级至三级之升级由副总监（监测与情报）批准；二级由总监批准；一级由总监或其授权之副总监批准，且不受程序限制——遇紧急情形，总值班室当值高级官员得先行发布一级预警、事后补办审批。
2. 预警等级之降级须比升级更为审慎——由引发升级之分析部门出具"态势缓解评估报告"，经相应审批层级确认后方可降级。
3. 同一时段内得针对不同区域或不同威胁类型发布不同等级之预警（例如：北方混沌前沿为橙炬，南方凡世态势为蓝焰）。
4. 一切预警之发布与降级均须留痕，记录发布时间、批准人、依据数据与降级理由。
