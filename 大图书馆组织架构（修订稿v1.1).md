## 一、万书塔

### 1.1 万书塔之形制

大图书馆之主体为**万书塔**——一座矗立于恰恩城中枢的白银巨塔，共**四百三十一层书架**，自地面拔起直入云端，为银色联盟最宏伟之建筑。

万书塔之形制仿自巴贝尔通天塔之下半部——当年天青骑士与布加人的祖先将通天塔一分为二，上半部与大图书馆同归湮灭，而万书塔正是布加人凭记忆与残留的建造法则，在恰恩重筑的致敬之作。塔身以银材与秘银合金为骨架，外壁镶嵌白色法则石，在日光下泛出冷冽的银辉，在夜间则因内部常年不灭之阅览灯火而通体温润如烛。

塔内四百三十一层书架自底层向上螺旋排列，每一层的书架沿塔壁环绕一周，中央为贯穿全塔之空井——仰望可见层层叠叠的书架向上延伸至视线尽头，恍若仰望一座由知识筑成的无底苍穹。空井中设有浮升术式平台，供馆员与访客在各层之间通行。

万书塔之最底层为大殿，即大图书馆之公共入口与中枢大厅；最高层为**星冕阁**——馆长之私人书房与馆务委员会之议事厅，从星冕阁之窗可俯瞰恰恩全城与云海。

万书塔之地下设有多层封存空间，以多重封印术式与常规楼层隔绝，用于保管最高安全等级之馆藏。

### 1.2 万书塔之圣殿属性

大图书馆不仅是一座图书馆，亦是银色联盟之**知识圣殿**。

白银之民以知识为信仰，以回收巴贝尔遗失之传承为天命。万书塔是这一信仰之物质化身——四百三十一层书架中的每一卷典籍，都是布加人数千年来守护文明火种之证明。对于白银之民而言，进入万书塔如同进入圣所，在此工作之馆员不仅是知识的管理者，亦是圣殿的守护者。

因此，大图书馆之馆员在联盟中享有特殊之敬意与称谓——他们被统称为**圣殿守护者**（Custodes Sacri），简称**守护者**。

## 五、七大典藏殿堂

> 万书塔四百三十一层并非均匀分配——各典藏殿堂依馆藏体量占据不同层数的区间，但彼此之间并非绝对隔断，而是以学科亲缘为序自下而上渐次排列。每一典藏殿堂既是物理空间（万书塔中的一段楼层），也是学术单元。殿堂主管为该领域之资深守护者，在馆务委员会中代表本殿堂发言。

| 事项 | 说明 |
|------|------|
| 层位 | 万书塔第一至第四十层 |
| 前身名称 | 编年长廊 |
| 馆藏 | 沃恩德历史、各族编年、战争记录、条约文献、重大事件卷宗 |
| 特色 | 万书塔最底层之四十层，是访客进入万书塔后首先经过之区域。白银之墙即位于永世回廊核心位置之第一层大殿。整个回廊以时间轴为序，从最古老之创世纪录到最近之历史事件，拾级而上，如同行走于白银之民的时间长河之中 |

## 七、守护者职级体系

### 7.1 圣殿守护者阶（学术序列）

大图书馆之守护者以学术为本，职级以"守护者阶"标识，自高至低共**七阶**。

| 阶 | 阶称 | 资质要求 | 对应岗位 | 备注 |
|----|------|---------|---------|------|
| 第一阶 | **至高守护者**（Custos Supremus） | 白银学会高级学术资质 + 百年知识工作经历 | 馆长 | 仅一人 |
| 第二阶 | **大守护者**（Custos Magnus） | 白银学会高级学术资质 | 副馆长、总馆务官 | |
| 第三阶 | **首席守护者**（Custos Primus） | 白银学会高级学术资质 + 卓越学术贡献 | 典藏殿堂主管、回收远征司主管、圣殿封存署主管 | |
| 第四阶 | **守护者**（Custos） | 白银学会基础学术资质，施法≥七环 | 资深馆员，独立主持研究项目或管理分室 | |
| 第五阶 | **副守护者**（Subcustos） | 白银学会基础学术资质，施法≥五环 | 独立承担编目、校勘、鉴定之馆员 | |
| 第六阶 | **学徒守护者**（Custos Initiatus） | 完成誓约登记 | 在守护者指导下工作之初级人员 | |
| 第七阶 | **见习者**（Novitius） | 联盟公民 | 各城邦学塔或银翼学院推荐之实习人员 | 完成见习后可晋升为学徒守护者 |

### 7.2 馆务序列（行政与安全）

馆务厅、圣殿封存署、编目与校勘署之行政与安全岗位采用独立序列。

| 级 | 级称 | 对应岗位 |
|----|------|---------|
| 一级 | **总馆务官** | 馆务厅主管（仅一人，同时为大守护者阶） |
| 二级 | **馆务长** | 署长、科长 |
| 三级 | **馆务官** | 室主任、资深行政人员 |
| 四级 | **馆务员** | 一般行政、安保、塔务维护人员 |

### 7.3 双轨说明

1. 守护者阶与馆务序列并行不交叉。典藏殿堂主管必须持守护者阶（首席守护者以上）。
2. 总馆务官为特殊情形——同时持有大守护者阶（学术身份）与馆务序列一级（行政身份），是连接两个序列之唯一桥梁。
3. 守护者阶之晋升依白银学会学术资质认定与馆务委员会评审。
4. 馆务序列之晋升依馆务厅之人事考核。
