# README
# 基于《琥珀之剑》的布加相关设定集

## 项目简介

本仓库为基于《琥珀之剑》世界观所衍生的非官方布加相关设定整理与创作项目，主要用于个人存档、设定整理与交流展示。

仓库内容包括但不限于：

- 对原作相关信息的整理与汇编
- 基于原作框架的个人扩写与补充设定
- 部分由作者撰写、修改或审校的 AI 辅助生成文本

本项目不代表原作官方立场，也不构成任何官方设定说明。

## 说明与范围

本仓库聚焦于布加相关内容，包括组织架构、人物关系、制度设定、历史补充、术语整理等。

为避免混淆，仓库中的内容大致分为以下几类：

- **原作整理**：对原作已有信息的摘录、归纳与重述
- **衍生扩写**：在原作基础上进行的非官方延伸设定
- **AI 辅助内容**：由 AI 生成、后经作者筛选、修改或整合的文本

如有需要，后续可在具体文件中继续标注内容来源与创作方式。

## AI 参与说明

本仓库部分文本包含 AI 辅助生成内容。AI 生成内容通常会经过人工筛选、改写、补充或校对，但仍可能存在：

- 与原作不完全一致的表达
- 设定冲突、遗漏或误读
- 风格不统一的问题

因此，仓库内容应视为非官方衍生设定文本，而非原作事实或官方资料。

## 仓库结构

示例：

- `docs/`：总说明、索引、术语表
