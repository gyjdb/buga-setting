# 银色舰队统帅部组织架构（修订稿 v2.1）

> **拟定依据：《银色联盟防务基本法》第五条至第十三条；《白银执政院机构设置》第0.5条**
> **性质：统帅部编制文件，规定银色舰队统帅部之完整内部架构，含首长层、内设机关、直属机构与设施、作战编制序列、单舰标准编制，以及对外接口。**
> **v2.0 → v2.1 修订说明：内设机构命名体系由“一厅十部”调整为“一厅七局三署”；统一统帅部内部层级语感，消除“部中有部”之重复；并对若干机关名称作军政化、制度化修饰。**

---

> 上位机关：白银议会（最高指挥权）
> 终局机关：银色学派（接管令之最高许可）
> 统帅部性质：议会直属·特许军事机关
> 后勤保障对口：白银执政院舰务行政部
> 独立监察对口：银烛会防务监察司

## 二、内设机关〔一厅七局三署〕

| ------ | -------------------------------------- |
| 秘书处    | 首长日程、批示流转、会务组织                         |

### 2. 联合作战局

舰队作战指挥之核心中枢。和平时期研拟战略与战役预案；战时为总帅之最高作战参谋班底，统一协调各舰队群之联合行动。

| 内设处室  | 职能                              |
| ----- | ------------------------------- |
| 战略规划处 | 联盟整体防务战略之研拟、长期兵力发展规划            |
| 战役筹划处 | 战役预案编制、战场选择与兵力部署方案              |
| 战术研发处 | 新战术研发、战法试验、战术教范编纂               |
| 演习统筹处 | 联盟级联合演习之策划、导调与评估                |
| 联合指挥处 | 战时联合作战指挥所之开设与运作、各舰队群之作战协调       |
| 征召编组处 | 战时征召城邦卫戍军之编组方案拟定与指挥衔接（以议会授权为前提） |

---

### 3. 战略情报局

舰队之军事情报中枢。独立于银烛会之情报体系——银烛会管节点监测与混沌态势，本局管军事层面之战场情报与威胁评估。

| 内设处室    | 职能                               |
| ------- | -------------------------------- |
| 态势研判处   | 汇总各方情报，形成综合战场态势图，为联合作战局提供研判支撑    |
| 空域监控处   | 联盟军事管控空域之实时监控，不明飞行物之识别与追踪        |
| 凡世军情处   | 凡世诸国之军事动向、兵力部署、舰队规模与军事技术之情报收集与分析 |
| 反间与安全处  | 防范外部势力对舰队之渗透与情报窃取，内部安全审查         |
| 混沌威胁评估处 | 接收银烛会混沌态势通报，独立评估混沌力量对舰队之军事威胁等级   |

---

### 4. 战备动员局

舰队编成、装备状态与动员准备之管理中枢。确保舰队在任何时刻具备规定之战备等级。

| 内设处室   | 职能                                  |
| ------ | ----------------------------------- |
| 编制处    | 年度编制表之拟定与调整（须报议会批准）、舰队群编成之优化建议      |
| 兵力配置处  | 灰袍巫师之征募、编入、跨舰队群调配与退役管理              |
| 装备统管处  | 全军舰只与武备之在册管理、装备状态追踪、报废核销与更新需求上报     |
| 战备等级处  | 战备等级之设定、检查与调整，全军战备值班制度之管理           |
| 动员规划处  | 三级动员预案之拟定与更新（戒备、局部、全面），城邦卫戍军征召之兵力测算 |
| 预备役管理处 | 退役灰袍巫师之预备役登记、定期复训与紧急召回体系            |

| 空域管控处 | 军事管控区之划定与出入许可管理（与天空航权与城体部之民用空域分界协调） |
| 航道安全处 | 航道上之海盗、魔物袭击与异常事件之应对预案，护航效能评估        |

### 11. 舰务技术署

舰只之战场抢修、日常维保与技术状态管理。此署为统帅部之“技术手臂”——与舰务行政部之区别在于：舰务行政部管船坞大修与建造，本署管舰队在役期间之日常维保与战场应急。

| 内设处室  | 职能                                     |
| ----- | -------------------------------------- |
| 舰体维保处 | 在役舰只之日常维护计划、定期检修之组织（小修归本署，大修送船坞归舰务行政部） |
| 战场抢修处 | 战时舰只损伤之应急修复方案、随舰损管队之编制与调度              |
| 结界维护处 | 舰载结界之日常校准、强度检测与性能衰减预警                  |
| 引擎保障处 | 舰载引擎之运行监控、故障诊断与应急处置                    |
| 技术状态处 | 全军舰只之技术状态档案管理、寿命预测与退役建议                |

## 五、作战编制序列

### 5.1 舰队群总览

银色舰队设六个常设舰队群与两个特别编组。各舰队群之驻泊空域覆盖联盟全部战略方向，确保任一方向遭受威胁时可在最短时间内投送兵力。

| 编号 | 名称 | 常驻空域 | 主要任务 | 典型编成 |
|------|------|---------|---------|---------|
| 第一舰队群 | **银心舰队群** | 卡奈奇—白塔—恰恩核心城邦群 | 联盟心脏之战略屏障，总帅直属战略预备队 | 分舰队×4（含1支主力分舰队） |
| 第二舰队群 | **冰脊舰队群** | 大冰川—格雷修斯方向 | 北方边境巡航，大冰川混沌前沿之威慑与封锁 | 分舰队×3 |
| 第三舰队群 | **星聚舰队群** | 星聚海—萨萨尔德上空 | 东南方向航道护航，星聚海沿线之安全维护 | 分舰队×3 |
| 第四舰队群 | **永黯舰队群** | 永黯之海—埃鲁因以南空域 | 南方纵深监视，凡世边境之最低接触缓冲屏障 | 分舰队×2（重巡逻编成） |
| 第五舰队群 | **云脊舰队群** | 云雾山脉—圣白草原上空 | 西部边境巡航，连接圣奥索尔方向之战略通道守护 | 分舰队×2 |
| 第六舰队群 | **极光舰队群** | 那泽尔—远北高空域 | 远北边境巡航，极端环境下之前沿存在，混沌渗透之第一道预警线 | 分舰队×2（轻型快速编成） |
| 特别编组·甲 | **遗产远征护航编队** | 依任务指定 | 大图书馆回收远征之专属护航力量，从各舰队群抽组 | 视任务规模编组 |
| 特别编组·乙 | **联合打击编队** | 依任务指定 | 战时跨舰队群之联合打击力量，由总帅直接指挥 | 视战役规模抽组 |
